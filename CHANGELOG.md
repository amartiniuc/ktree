## [0.1.0] - 2025-01-18

### Added
- Initial release of KTree
- Full-screen CLI application for browsing Kubernetes clusters
- Tree-based navigation with vim-style keybindings
- Browse namespaces → object types → objects → details
- View object details (Describe view)
- View pod logs with auto-scroll
- Exec commands with clipboard copy functionality
- Search/filter functionality with visual indicators
- Help overlay (CTRL+B) with all keybindings
- Dynamic column width adjustment
- Cascading selection on startup
- Async data loading with background workers
- Loading and empty states
- Cross-platform binary builds (macOS, Linux, Windows)
- Comprehensive documentation
